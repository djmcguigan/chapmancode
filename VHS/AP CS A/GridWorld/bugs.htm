GridWorld Framework Code Errata and Changes

This page lists errata and changes in the code for the GridWorld framework code. For errata in the narrative, please see http://lauriewhite.org/AP/. The latest snapshot of the GridWorld framework code is available at http://horstmann.com/gridworld/GridWorldCode.zip. Note that this code is more up-to-date than the code at the College Board site, so you should check it first.

The following issues have been addressed in version 0.97

  • Postconditions have been added to methods of the Critter class.
